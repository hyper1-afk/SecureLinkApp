Important: App Password Required

Most email providers require an App Password instead of your regular password. Select your email provider below:

For Gmail (@gmail.com):

  1. Go to your Google Account Security
  2. Enable 2-Step Verification (required)
  3. Go to App Passwords
  4. Select "Mail" and "Windows Computer", then click Generate
  5. Copy the 16-character password and use it below

IMAP Server: imap.gmail.com | Port: 993

For Outlook/Hotmail (@outlook.com, @hotmail.com, @live.com):

  1. Go to Microsoft Account Security
  2. Click "Advanced security options"
  3. Enable Two-step verification (required)
  4. Under "App passwords", click "Create a new app password"
  5. Copy the generated password and use it below

IMAP Server: outlook.office365.com | Port: 993

Microsoft Help Guide →

For Yahoo (@yahoo.com):

  1. Go to Yahoo Account Security
  2. Enable Two-step verification (required)
  3. Click "Generate app password"
  4. Select "Other App" and enter "SecureLink"
  5. Copy the generated password and use it below

IMAP Server: imap.mail.yahoo.com | Port: 993

Yahoo Help Guide →
