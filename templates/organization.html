Integrate with Your Network

Point any tool, script, or network device at the gateway endpoint using your org API key. Returns ALLOW/BLOCK instantly.

Request

curl -X POST https://securelinkapp.com/api/gateway/check \
  -H "X-API-Key: YOUR_ORG_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://suspicious-site.com", "user_id": 42}'

Response

{
  "verdict": "block",
  "reason": "Risk score 0.85 exceeds threshold 0.50",
  "risk_score": 0.85,
  "risk_level": "high",
  "threats": ["phishing"],
  "log_id": 1042
}
